[winterjs](https://github.com/wasmerio/winterjs)

WinterJS 是一个用 Rust 编写的 JavaScript Service Workers 服务，使用 SpiderMonkey 运行时。 它速度快，与其他服务兼容，并且可以编译为 WebAssembly。 WinterJS 允许在各种环境中运行 Service Worker。
